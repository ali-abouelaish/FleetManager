=====================================
  QUICK GUIDE: Get Google Maps API Key
=====================================

STEP 1: Go to Google Cloud Console
‚Üí https://console.cloud.google.com/

STEP 2: Create Project
‚Üí Click "Select a project" (top of page)
‚Üí Click "NEW PROJECT"
‚Üí Name: "Fleet Management"
‚Üí Click "CREATE"

STEP 3: Enable APIs (Do this TWICE)
‚Üí Left menu: "APIs & Services" ‚Üí "Library"
‚Üí Search: "Maps JavaScript API"
‚Üí Click it ‚Üí Click "ENABLE"
‚Üí Go back to Library
‚Üí Search: "Geocoding API"
‚Üí Click it ‚Üí Click "ENABLE"

STEP 4: Create API Key
‚Üí Left menu: "APIs & Services" ‚Üí "Credentials"
‚Üí Click "+ CREATE CREDENTIALS" (at top)
‚Üí Select "API key"
‚Üí COPY THE KEY! (looks like: AIzaSyB1234...)

STEP 5: Add to Your Project
‚Üí Open file: D:\Fleet\.env.local
‚Üí Add this line:

NEXT_PUBLIC_GOOGLE_MAPS_API_KEY=paste_your_key_here

(Replace "paste_your_key_here" with the actual key)

STEP 6: Restart Dev Server
‚Üí In terminal, press Ctrl+C to stop
‚Üí Type: npm run dev
‚Üí Press Enter

STEP 7: Test It!
‚Üí Go to: http://localhost:3000/dashboard/schools
‚Üí You should see the map! üó∫Ô∏è
